## Описание проекта

Анализ пользовательского взаимодействия с карточками статей сервиса Яндекс.Дзен.Строим дашборд  с помощью Tableau Public и делаем презентацию.

## Навыки и инструменты

- **python**
- **pandas**
- **PostgreSQL**
- **Tableau**
- **sqlalchemy**
- **продуктовые метрики**
- **построение дашбордов**
